      ID

      Unique ID for joining boundary data. This column must contain values that match ID to match values in the Regions geometry sheet

      Accepted column types

      Name

      The name that appears in popups and panels

      Accepted column types

      Label

      The name that appears in region labels on the map

      Accepted column types

      Color by

      One or more values or categories to color your regions by. If this column is numeric, your regions will be shaded with a sequential or diverging scale. If it's categorical, your regions will be shaded with a categorical scale.

      Accepted column types

      Group

      Draw borders around groups of regions, e.g. state borders around counties

      Accepted column types

      Info for popups

      Makes additional columns (text, image URLs, embedded charts etc) available in custom popups

      GeoJSON geometry that describes a boundary (typically, geometry type will be MultiPolygon or Polygon)

      Accepted column types

      IDs to match

      One or more columns containing unique IDs for joining boundary data. This column must contain values that match ID values in the Regions sheet.
